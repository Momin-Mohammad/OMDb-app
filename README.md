# OMDb-app
A simple app to show movie data using omdb api.

## Deployed link:
[omdb-app](https://omdb-dummy.netlify.app/)

## Features:
1. Search suggestion
2. pagination
3. data display
4. proxy server

## NOTE: THE SITE TAKES SOME TIME TO LOAD DATA. PLEASE REFRESH THE SITE IF NO DATA IS DISPLAYED
